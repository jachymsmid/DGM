Detailed Description

template<typename Real = double, typename Device = TNL::Devices::Host, typename Index = int>
class DG::Integrator< Real, Device, Index >

Abstract base for time integrators advancing the semi-discrete DG system.

Subclasses implement step() and integrate(). Integrators operate on FieldVector instances and accept an RHS callback describing the spatial operator.
