Detailed Description

Time integrators for advancing the semi-discrete DG system.

Declares an abstract Integrator interface and several concrete explicit Runge–Kutta schemes used by the solver (ERK, LSERK, SSPRK). Integrators accept an RHS callback and operate on FieldVector instances.
