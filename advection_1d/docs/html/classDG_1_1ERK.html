Detailed Description

template<typename Real = double, typename Device = TNL::Devices::Host, typename Index = int>
class DG::ERK< Real, Device, Index >

Classic explicit four-stage Runge–Kutta integrator (ERK4).

Implements a straightforward RK4 integration using temporary stage storage sized by K * Np.
